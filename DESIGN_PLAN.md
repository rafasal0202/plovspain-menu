# PlovSpain Menu - Дизайн и Архитектура

## Цель
Создать современный, технологичный веб-интерфейс для удобного формирования заказов узбекской кухни, где пользователь может:
- Видеть все блюда в красивом, современном формате
- Легко выбирать количество и видеть стоимость в реальном времени
- Оформить заказ в один клик через WhatsApp/Telegram

## Архитектура Интерфейса

### 1. Главная страница (Menu Grid)
- **Макет:** Сетка блюд (grid 2x2 или 3x3 в зависимости от экрана)
- **Каждое блюдо:** Карточка с фото, названием, описанием, ценой
- **Интерактивность:** Клик на карточку → открывается модальное окно с выбором количества
- **Дизайн:** Минималистичный, современный, с хорошей типографикой

### 2. Модальное окно выбора (Product Modal)
- **Содержимое:**
  - Большое фото блюда
  - Название и описание
  - Цена за единицу
  - Минимальное/максимальное количество
  - Кнопки +/- для выбора количества
  - Кнопка "Добавить в корзину" (с отображением итоговой стоимости)
- **Дизайн:** Чистый, с хорошей иерархией информации

### 3. Корзина (Shopping Cart)
- **Расположение:** Фиксированная панель внизу экрана (или плавающая кнопка)
- **Содержимое:**
  - Список выбранных блюд с количеством
  - Итоговая стоимость
  - Кнопка "Оформить заказ"
- **Дизайн:** Компактный, но информативный

### 4. Форма оформления заказа (Checkout) - МНОГОШАГОВАЯ

#### Шаг 1: Выбор способа доставки
- **Два варианта:**
  - 🚗 **Доставка** - требует ввод адреса
  - 🏪 **Самовывоз** - без адреса
- **Дизайн:** Две большие кнопки/карточки на выбор

#### Шаг 2: Адресные данные (ТОЛЬКО если выбрана доставка)
- **Поля:**
  - Улица (обязательно)
  - Номер дома (обязательно)
  - Этаж (обязательно)
  - Квартира (обязательно)
- **Валидация:** Все поля должны быть заполнены перед переходом
- **Дизайн:** Четкие input-поля с подсказками

#### Шаг 3: Выбор канала связи
- **Два варианта:**
  - 💬 **WhatsApp** (+34602478357)
  - 📱 **Telegram** (@salievr)
- **Дизайн:** Две большие кнопки с иконками

#### Шаг 4: Подтверждение и отправка
- **Показываем:**
  - Список блюд с количеством
  - Итоговую стоимость
  - Способ доставки (Доставка/Самовывоз)
  - Адрес доставки (если применимо)
  - Канал связи (WhatsApp/Telegram)
- **Кнопка:** "Отправить заказ" - открывает WhatsApp/Telegram с готовым сообщением

## Цветовая Схема
- **Основной цвет:** Теплый красный/оранжевый (узбекская кухня)
- **Акцентный цвет:** Золотой/желтый (богатство, праздничность)
- **Фон:** Белый или светло-серый (чистота, современность)
- **Текст:** Темно-серый/черный (читаемость)

## Типография
- **Заголовки:** Жирный, современный шрифт (Montserrat, Inter, Poppins)
- **Основной текст:** Чистый, читаемый шрифт (Open Sans, Roboto)
- **Иерархия:** 3-4 уровня размеров для четкой структуры

## Современные элементы

- ✅ Микровзаимодействия (hover-эффекты, feedback)

- ✅ Четкие CTA-кнопки
